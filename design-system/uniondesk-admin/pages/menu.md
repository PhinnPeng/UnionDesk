# 菜单页覆盖

> **项目：** UnionDesk Admin
>
> **页面类型：** 导航配置 / 菜单管理

> 本文件中的规则优先于 `design-system/uniondesk-admin/MASTER.md`。这里只写菜单页特例。

---

## 页面级规则

- 页面采用「树形菜单 + 详情弹窗」的后台导航维护布局。
- 左侧树要清晰显示目录、菜单、按钮三层结构，右侧编辑弹窗负责新增与修改。
- 权限码下拉必须按 `platform / domain / shared` 分组，便于管理员区分权限域。
- 图标选择器要保持可视化，不要退回纯文本输入。
- 节点名称、路由、组件、权限码要一眼可见。
- 新建和编辑尽量复用同一套表单。
