# 域配置页覆盖

> **项目：** UnionDesk Admin
> **页面类型：** 域级设置 / 配置中心
> **生成时间：** 2026-05-06

> 本文件中的规则优先于 `design-system/uniondesk-admin/MASTER.md`。除本页特例外，其余规则均以 Master 为准。

---

## 页面级规则

### 布局

- 使用域信息头 + 分组表单 + 固定保存条的设置布局。
- 页面最大宽度 1440px，表单分组要清晰，便于域级配置维护。

### 结构

- 顶部展示域名称、编码、状态和基础信息。
- 中部按模块分组配置开关、文本、阈值、默认值等。
- 底部或右下角提供统一保存与撤销操作。

### 间距

- 分组卡片保持适中间距，不要像营销页一样留出过大空白。

### 字体

- 配置标题清晰，字段说明使用辅助字号。

### 颜色

- 使用中性底色与明确的开关颜色，不做装饰性渐变。

### 组件

- 使用表单分组、开关、输入框、说明文本、保存条。

### 避免

- 避免 hero、case study、sponsor 等站点模板。
- 避免把配置页做成单纯详情展示页。
