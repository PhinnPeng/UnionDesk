# Domains Override

## Page Intent

- Customer business-domain selection and invitation-code onboarding.
- The page should help users enter or switch to the correct support domain quickly.

## Required Structure

- Domain card grid with clear selection states
- Invitation code entry or join action
- Domain metadata presented in a compact way

## Visual Rules

- Keep the page simple, operational, and portal-like.
- Avoid enterprise gateway marketing language.
- Use short labels such as 选择业务域、加入业务域、当前业务域.

## Interaction Rules

- Selected domain states should be obvious.
- Support empty, loading, and invitation error states.
