# Inbox Override

## Page Intent

- Customer notification center for ticket progress, system notices, and security alerts.
- This is a message handling surface, not a content feed.

## Required Structure

- Notification list with read / unread distinction
- Category filters
- Mark-as-read action
- Clear empty state when no messages exist

## Visual Rules

- Keep list rows compact and scannable.
- Preserve enough height for empty state so the panel does not look collapsed.
- Use restrained badges and status tags.

## Interaction Rules

- Support quick read and navigation to the related ticket or detail page.
