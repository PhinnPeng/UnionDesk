# 工单能力收口清单

## 业务域
- 工单

## 目标
- 将工单重开、合并、模板、SLA、优先级、批量操作、附件限制和客户权限收口到 PRD，并确保数据模型描述一致。

## 开发任务
- [x] 明确已关闭工单不支持重开，继续处理时只能新建工单并关联原单。
- [x] 明确客户撤回仅允许在待分配 / 待处理阶段，已解决后不可追加回复。
- [x] 明确重复工单可合并到主单，被合并单进入终态且 SLA 立即停止。
- [x] 明确工单模板与内容模板分层，前者用于内部快速创建，后者用于客户手动应用。
- [x] 明确 SLA 违约动作支持复选，且自动转给指定人员采用协助 / 抄送模式。
- [x] 明确优先级可增减并支持域级自定义显示文案。
- [x] 明确工单队列支持批量领取、批量指派、批量关闭和批量导出。
- [x] 明确附件默认白名单覆盖常见格式，域管理员可继续增补。
- [x] 补充 `ticket_template`、`ticket_relation`、`ticket_priority_level`、`attachment_policy` 与 `sla_rule.breach_action_json`。
